Competências

Código é minha linguagem

JavaScript

Uma linguagem de programação versátil e poderosa, é a força motriz por trás da interatividade e dinamismo da web moderna.

Python

Simplicidade, legibilidade e versatilidade em uma linguagem de programação de alto nível, ideal para uma ampla gama de aplicações.

Flask

Micro-framework web em Python, simples e flexível, ideal para criar rapidamente APIs RESTful e aplicativos web dinâmicos.

Html

A linguagem de marcação essencial para criar páginas web, estruturando conteúdo e fornecendo a base para experiências online envolventes.

CSS

a linguagem de estilo que dá vida à web, moldando o visual e a apresentação de páginas e aplicativos online.

MYSQL

Sistema de gerenciamento de banco de dados relacional confiável e amplamente utilizado, fundamental para armazenar e gerenciar dados em aplicativos web e de negócios.

React Native

Um framework de desenvolvimento ágil de apps móveis para iOS e Android, baseado em JavaScript.

Figma

Plataforma colaborativa e intuitiva para design de interfaces de usuário, simplificando o processo de criação de protótipos e designs digitais.
